Сборка через GitHub (без Android Studio)

1) Создай новый пустой репозиторий на GitHub.
2) Загрузи ВЕСЬ проект из этой папки (все файлы и папки).
3) Открой вкладку Actions.
4) Выбери workflow "Build APK".
5) Нажми "Run workflow".
6) После завершения открой Artifacts -> smsbridge-debug-apk и скачай APK.

APK будет в артефакте GitHub Actions.
